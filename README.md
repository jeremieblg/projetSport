# Projet Sport

## Prerequi
```
npm install -g @vue/cli
npm install -g json-server
```
### Lancer le faux server
```
json-server .\db.json
```

## Installer le projet
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
